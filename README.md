# nomingbai

A backend-first web project scaffold.

## Project structure

- `src/server.js` - Express server entrypoint
- `src/routes/index.js` - HTTP routes
- `src/controllers/homeController.js` - request handlers
- `public/index.html` - simple web UI served by the backend

## Run locally

```bash
npm install
npm run dev
```

Then visit `http://localhost:3000`.
